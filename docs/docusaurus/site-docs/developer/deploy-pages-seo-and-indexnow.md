# Pages SEO & IndexNow

The docs site is intended to publish to GitHub Pages under the repository project path:

- site URL: `https://nick2bad4u.github.io/eslint-plugin-docusaurus-2/`
- sitemap: `/sitemap.xml`
- IndexNow key file: `/indexnow-key.txt`

Keep those URLs aligned with `docusaurus.config.ts`, `robots.txt`, and the IndexNow helper scripts/tests.
